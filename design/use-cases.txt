There are two types of users:
1) Person
2) IoT device

1) Person
The customer without permission can:
- login with email and password
- reset forgotten password (*) (implement email system)
- change password (from "account" section)
- interact with IoT devices
	- switch on/off
	- set schedule or timer (schedule is set on the server or the IoT device)
	- check connection

The privileges are:
- manage virtual house accounts
	- invite new user
	- remove user
- giving permissions 
- manage IoT device
	- add IoT device
	- remove IoT device
	- modify IoT device

2) IoT Device
The IoT device can:
- connect to the server
- send data
- receive data
