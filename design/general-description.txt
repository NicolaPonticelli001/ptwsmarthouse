There are two subdomain:
- management.ptwsmarthous.it: this is the subdomain reserved for the operators that will
manage the customers account;
- customer.ptwsmarthouse.it: this is the subdomain reserved for the customers who will
manage the IoT device with.
The customer's website is general for everyone but the content will change according to the user
(dynamic website). There will be one applicative role for the user and any user can manage the
IoT devices.
